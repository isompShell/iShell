1.本服务菜单是用来开启或关闭堡垒主机系统相关服务的开关。
2.执行menu运行本服务菜单或增加执行权限运行此脚本
3.1-10为检测服务开启状态菜单，输入每条服务前面数字，即进入选择服务的开启与关闭配置
4.输入man进入帮助界面，根据提示，进行相关配置
5.进入服务后，输入开关前面数字，即可开启或关闭当前服务，输入0则返回上一层菜单
6.如有输入错误，会提示错误输出，3秒后返回当前菜单
7.如输入有误，请按shift+Backspace删除错误信息

1 the service menu is used to open or close the fortress host system related services switch.
2 perform menu to run the service menu or increase the execution permissions to run this script
3.1-10 to detect the service open state menu, enter the number of each service in front of the number. that is, to enter the choice of service to open and close the configuration
4 input man to enter the help interface, according to the prompts, to carry out the relevant configuration
5 after entering the service, enter the switch in front of the number, you can turn on or off the current service, enter the 0 return on the first layer of the menu
6 if there is an input error, will prompt the error output, 3 seconds after the return to the current menu
7 if the input is incorrect, please press shift+Backspace to delete the error message.
